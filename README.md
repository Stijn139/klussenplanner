# 🔨 Klusplanner

Een persoonlijke klusplanner voor het bijhouden en plannen van klussen en verbouwingsprojecten.

## Features

- **Kanban-bord** — Taken beheren in kolommen: Te doen → Bezig → Klaar
- **Projecten** — Grotere klussen opsplitsen in subtaken (bijv. Voordeur, Tuin)
- **Planning** — Tijdlijn-overzicht van alle geplande klussen
- **Prioriteiten** — Hoog / Midden / Laag met kleurcodering
- **Datum & Tijd** — Plan klussen in met datum en optioneel tijdstip
- **Google Agenda export** — Download .ics bestanden om te importeren in Google Calendar
- **Fullscreen modus** — Gebruik de ⛶ knop voor volledig scherm
- **Offline** — Werkt volledig in de browser, geen server nodig

## Gebruik

Open `index.html` in je browser, of bezoek de live versie via GitHub Pages.

Alle data wordt lokaal opgeslagen in je browser (localStorage).

## .ics Export naar Google Calendar

1. Plan een klus in met datum (en optioneel tijdstip)
2. Klik op 📅 bij de taak of gebruik "Exporteer alles (.ics)"
3. Open het gedownloade `.ics` bestand, of importeer via Google Calendar → Instellingen → Importeren

## Deployment

Deze app draait als statische HTML pagina via [GitHub Pages](https://pages.github.com/).
